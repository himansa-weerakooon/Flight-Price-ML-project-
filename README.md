# Flight Price Prediction

This project focuses on analyzing flight booking data to understand how ticket prices fluctuate in the aviation market of India.

The process involved comprehensive Exploratory Data Analysis (EDA) and Advanced Data Analysis (ADA) to handle the dataset.
The process involved comprehensive data cleaning and preprocessing to handle inconsistencies in the dataset. Various models were built and the best was chosen to ensure accurate results.
We specifically examined the differences between Business and Economy class bookings to see how they affect costs.

Source: [Kaggle - Flight Price Prediction](https://www.kaggle.com/datasets/shubhambathwal/flight-price-prediction)
